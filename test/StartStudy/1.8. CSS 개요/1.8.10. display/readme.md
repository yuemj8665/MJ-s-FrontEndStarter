
6. disply
- 요소의 화면 출력(보여짐) 속성
- 각 요소에 이미 지정되어 있는 값
    - block : 상자(레이아웃) 요소. (div)
    - inline : 글자 요소 (span)
    - inline-block : 글자 + 상자 요소 ()
- 따로 지정해서 사용하는 값'
    - flex : 플렉스 박스(1차원 레이아웃)
    - grid : 그리드 (2차원 레이아웃)
    - none : 보여짐 특성 없음, 사라짐.
- 기타 : table, table-row, table-cell 등등..
- 정리 : HTML Inline 요소에 가로너비를 인위적으로 변경하기 위해서는 display: block;
같은 요소를 사용한다.
