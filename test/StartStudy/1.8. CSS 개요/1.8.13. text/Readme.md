### 7.2.6. 문자
- color : 글자의 색상
- RGB(0,0,0) : 기본 값, 검정색
- 색상 : 기타 지정 가능한 색상

- text-align
    - left : 기본값 왼쪽정렬
    - right
    - center 등등..
- text-decoration
    - none : 장식 없음
    - underline : 밑줄
    - overline : 윗줄
    - line-throught : 취소선
- text-indent
    - 0 : 기본값, 들여쓰기 없음.
    - 양수입력 : 들여쓰기
    - 음수입력 : 내어쓰기
